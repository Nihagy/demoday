Sobre

O Projeto tem como objetivo ajudar mulheres que sofrem algum tipo de abuso e assim oferecer um serviço para mulheres de diferentes idades, raças e crenças.

Teremos no nosso espaço reuniões com debates, palestras, sempre acompanhando, orientando e motivando. Tudo isso com um espaço que transmite paz, e de confiança em que pode buscar ajuda como nosso sistema SAD(Sistema Anônimo de Denúncia).

Software

Software

Para essa aplicação especifica, usaremos esse software para a base de nosso sistema. Nosso foco é abordar o assunto da violência contra a mulher, além de mostrar a localidade para os usuários do sistema. Nosso sistema, inicialmente, terá duas opções: “denúncia de abusos” e “sobre o local”.
Podendo assim fazer uma denúncia, ou apenas aprender mais sobre o lugar que está localizado o totem. Para uma mulher se relacionar com o sistema ele tem duas opções, se cadastrar ou permanecer anônima. Sendo que ambas as entradas serão contabilizadas no banco de dados para realizar calculos estastistico. No banco de dados, já que citamos, usaremos a linguagem SQL, que por ser opensource, tem uma boa compatibilidade com sistemas Linux.
